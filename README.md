# 📊 Análise de Cancelamento de Clientes (Customer Churn)

Projeto de análise exploratória de dados utilizando **Python**, **Pandas** e **Plotly** para identificar os principais fatores relacionados ao cancelamento de clientes (Customer Churn).

## 🎯 Objetivo

O objetivo deste projeto é analisar uma base de dados de clientes para responder à seguinte pergunta:

> **Quais características estão mais associadas ao cancelamento de clientes e quais ações podem ser tomadas para reduzir esse índice?**

Durante a análise são realizadas etapas de limpeza, tratamento dos dados, visualização gráfica e interpretação dos resultados.

---

## 🛠️ Tecnologias utilizadas

* Python 3
* Pandas
* Plotly Express
* Jupyter Notebook

---

## 📋 Etapas da análise

* Importação da base de dados;
* Limpeza e tratamento dos dados;
* Remoção de informações irrelevantes;
* Análise exploratória (EDA);
* Criação de gráficos interativos com Plotly;
* Identificação dos fatores relacionados ao cancelamento de clientes;
* Geração de insights para auxiliar na tomada de decisão.

---

## 📈 Visualizações

O projeto utiliza gráficos interativos do Plotly para analisar a distribuição das variáveis em relação ao cancelamento dos clientes.

Exemplos de análises realizadas:

* Tempo de contrato
* Tipo de assinatura
* Quantidade de ligações para suporte
* Forma de pagamento
* Faixa etária
* Demais variáveis presentes na base de dados
